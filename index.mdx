## About the System

The Cloro de Hidalgo Procurement System is a comprehensive enterprise solution designed to optimize the supply chain workflow for raw materials and supplies. Built with Next.js and Supabase, it enables seamless communication between Purchasing, Laboratory, CEDIS departments, and external suppliers.

## User Roles

The system supports multiple user roles with specific permissions:

<AccordionGroup>
  <Accordion title="Admin" icon="user-shield">
    Full system access including user management, catalog administration, and system configuration
  </Accordion>
  <Accordion title="Coordinator" icon="user-tie">
    Manage requisitions, communicate with suppliers, and coordinate deliveries
  </Accordion>
  <Accordion title="Laboratory" icon="flask">
    Technical validation and quality control monitoring
  </Accordion>
  <Accordion title="CEDIS" icon="warehouse">
